プロジェクト　仮）しおり計画
======================

概要
------
旅行やイベントに行く人向けのスケジュール計画アプリ
（次に追加する機能として、待ち合わせアプリを追加、イベント等の主催者向けにユーザーなどに情報発信するサイト作成機能を想定）

利用対象
------
旅やイベントによく行く20代から40代の男女を対象に

機能
------
・イベント等の情報を記録管理
・スケジュール管理
・地図情報(位置情報表示)
・交通
・メモ
・アラーム機能
・同行者との情報共有(メールなどより情報ページの共有)
・情報をみんなに公開できる。
・アラーム機能

開発対象言語
------

### 使用言語 ###
・MySQL
・PHP
・HTML
・CSS
・JavaScript

### 対象API ###
・Google Map API or Bing Map  API
・ぐるなびAPI


対象デバイス
------

PC、スマートフォン(iOS、Android)
